# README

This is a Rails Project on a facebook like social media. It has most of the functionalities of facebook except realtime chat.

Notes:

* Ruby version - 2.6

* System dependencies - Devise, Omniauth

* Database - Postgresql

Done by:- Daniel  Mitiku
